# Changelog

All notable changes to Rustisaur will be documented in this file.

## [0.1.0] - 2026-06-21

### Added

- Initial Rustisaur engine with Lua 5.4 integration via mlua
- Core engine with Tokio async runtime
- Standard library: console I/O, file I/O, HTTP client, JSON, events
- Security sandbox with memory and instruction limits
- CLI with `run`, `repl`, `check`, and `version` commands
- Event system with broadcast-based emitters
- Plugin system architecture (in-process plugins)
- C FFI layer for embedding
- Examples and integration tests
- Documentation
